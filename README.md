# UnsupervisedCLEstimation

This repository contains the supplementary codes for the paper titled "Cognitive Load Estimation Using a Hybrid Cluster-Based Unsupervised Machine Learning Technique". For this research we used the CogWear dataset (https://physionet.org/content/consumer-grade-wearables/1.0.0/).

If you find this work helpful, consider citing the paper (https://doi.org/10.1109/access.2024.3428691).
